
GFG :

Given a string, we have to find out all subsequences of it. A String is a subsequence of a given String, that is generated by deleting some character of a given string without changing its order.
Examples: 

Input : abc
Output : a, b, c, ab, bc, ac, abc

Input : aaa
Output : a, aa, aaa



Recursive approach:


#include <bits/stdc++.h>
using namespace std;

void recur(string input, string output)
{
    
    if (input.empty()) {
        cout << output << endl;
        return;
    }
    recur(input.substr(1), output + input[0]);
    recur(input.substr(1), output);
}
int main() {
    string str;cin>>str;
    int n=str.size();
    recur(str,"");
	return 0;
}
